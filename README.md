# Geneticaust
Evaluación Comparativa de Algoritmos  Genéticos y Métodos Exactos

## Requisitos del Sistema

- Sistema operativo: Windows
- Python 3.8 o superior
- Visual Studio Code (VS Code)

## Extensiones necesarias en VS Code
1. Python (de Microsoft)
2. Jupyter (de Microsoft)

Para instalar:
- Abre VS Code
- Ve a la pestaña de extensiones (ícono de cuadrados en la barra lateral izquierda)
- Busca e instala las extensiones: `Python` y `Jupyter`
